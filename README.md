# Dashboard de Performance de Cobrança 📊

Este projeto é um dashboard interativo desenvolvido para o monitoramento e análise da performance de uma equipe de cobrança. O painel apresenta métricas visuais claras, permitindo acompanhar o volume de acionamentos e os resultados alcançados.

## 🚀 Acesso ao Dashboard
Você pode visualizar o dashboard em funcionamento online através do link abaixo:

👉 **[Acesse o Dashboard Aqui](https://lunvicks.github.io/dashboard_itau/)**

## 📊 Principais Funcionalidades
- **KPIs em Destaque:** Visualização rápida do total de acionamentos, operadores ativos e principais resultados.
- **Gráficos Dinâmicos:** - Comparativo de acionamentos por operador.
    - Distribuição percentual por canais de atendimento (WhatsApp, Telefone, SMS, E-mail).
    - Análise dos resultados de cobrança (Promessas de pagamento, recusas, etc.).
    - Evolução temporal dos dados ao longo do mês.
- **Design Responsivo:** Adaptado para diferentes resoluções de tela.

## 🛠 Tecnologias Utilizadas
- **HTML5 & CSS3:** Estrutura e estilização moderna.
- **JavaScript (ES6+):** Lógica e interatividade.
- **Chart.js (v4.4.1):** Biblioteca para renderização de gráficos de alta performance.
