# Dashboard de Performance de Cobrança 📊

Este é um projeto de dashboard interativo para monitoramento de acionamentos da equipe de cobrança, construído com HTML, CSS e JavaScript, utilizando a biblioteca **Chart.js**.

## 🚀 Funcionalidades
- **KPIs em tempo real:** Visão consolidada de acionamentos e operadores ativos.
- **Gráficos Dinâmicos:**
  - Acionamentos por operador.
  - Distribuição percentual por canais.
  - Performance por tipo de resultado.
  - Evolução temporal dos dados.

## 🛠 Tecnologias Utilizadas
- HTML5 & CSS3
- JavaScript (ES6+)
- Chart.js (v4.4.1)

## 💻 Como rodar
1. Clone este repositório: `git clone https://github.com/Lunvicks/dashboard_itau.git`
2. Abra o arquivo `index.html` em qualquer navegador moderno.
